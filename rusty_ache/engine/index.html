Main engine module defining engine traits and implementation.

Provides abstractions for engine lifecycle and rendering, as well as a concrete GameEngine implementation which manages rendering, scene management, input handling, and the event loop.

This module integrates configurations, scenes, rendering, and input processing to provide the core game engine loop and functionality.
