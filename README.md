# rusty_ache
Blazingly fast game engine written in Rust.

## How to run
Clone this repo:
```bash
  git clone git@github.com:WoWaster/Geraltt.git
```
Open project:
```bash
  cd spbu-tt-diff-notify
```
Run project:
```bash
  cargo run --bin main
```
